# Coursera Subtitle Toggle (C key)

Press **C** to toggle subtitles on/off during Coursera video lectures.

## Install

1. Install [Tampermonkey](https://www.tampermonkey.net/)
2. [Click here to install](../../raw/main/coursera-subtitle-toggle/coursera-subtitle-toggle.user.js)

## How it works

- Toggles all text tracks on the `<video>` element
- Prefers English subtitles when enabling
- Ignores keypresses in text inputs
