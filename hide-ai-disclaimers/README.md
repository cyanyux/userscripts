# Hide AI disclaimers

Hide disclaimer containers on ChatGPT, Gemini, and Claude.

## Install

1. Install [Tampermonkey](https://www.tampermonkey.net/)
2. [Click here to install](https://github.com/cyanyux/userscripts/raw/main/hide-ai-disclaimers/hide-ai-disclaimers.user.js)

## Supported sites

- `chatgpt.com`
- `chat.openai.com`
- `gemini.google.com`
- `claude.ai`
